# DSP-Calculator

戴森球计划量化工具网页版

## 使用说明
该项目基于Flask制作，所以
- 对于懂点python的，`pip install Flask`装上Flask，本地运行即可。
- 对于小白来说，可以等哪位大佬分享出来一个服务器直接用就好
